# LCSS
## 最長共同子序列演算法
最長共同子序列(LCS)常被應用於DNA序列比對或是版本控制等領域，請依照老師上課的投影片，實作出LCS演算法，並計算出兩個序列的最長共同子序列的長度
## Input
首先第一行輸入一個整數n為測資筆數，接下來第二行開始，每兩行字串(s1, s2)為一筆測資，請計算s1, s2的最長共同子序列的長度。
## Output
輸出最長共同子序列的長度
### Sample input
3
aabbacd  
aabacab  
mississippi  
ispim  
1123123  
111233 
### Sample output
5  
4  
5
